# ✅ My Tasks — To-Do App

A clean and functional To-Do application built with HTML, CSS & JavaScript that helps you manage your daily tasks efficiently.

## 🚀 Live Demo
[View Live Site](https://fatimarashid15.github.io/DecodeLabs-Project-3/)

## 📱 About
My Tasks is a simple yet powerful to-do app where you can add, complete, and delete tasks. It shows a live count of remaining tasks and works smoothly on all devices.

## ✨ Features
- ➕ Add new tasks instantly
- ✅ Mark tasks as complete
- ✕ Delete tasks
- 📊 Live task counter
- ⌨️ Press Enter to add tasks
- 📱 Fully responsive design

## 🛠️ Technologies Used
- **HTML5** — Semantic structure
- **CSS3** — Flexbox & Media Queries
- **JavaScript** — DOM Manipulation & Event Listeners

## 📂 How to Run
1. Clone or download this repository
2. Open `index.html` in any browser
